# XLSX Quality Checks

Check the workbook with an available spreadsheet library. Use LibreOffice recalculation when cached formula values must be updated.

Verify:

- expected worksheets exist and retain meaningful names;
- formulas reference the intended ranges;
- numeric, date and text cells keep appropriate formats;
- tables remain readable without accidental blank rows or columns;
- workbook output opens without repair warnings.

State clearly when formula recalculation could not be performed on the host.

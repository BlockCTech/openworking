# DOCX Quality Checks

Use an available office renderer such as LibreOffice when layout matters. If rendering is unavailable, inspect the package with a DOCX library and disclose the limitation.

Check:

- headings, lists and tables use consistent styles;
- page breaks and section breaks are intentional;
- headers, footers and page numbers remain intact;
- tables fit within page margins;
- no content is clipped, duplicated or left as placeholder text.

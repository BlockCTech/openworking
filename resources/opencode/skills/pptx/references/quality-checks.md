# PPTX Quality Checks

Use available tools such as LibreOffice to render a presentation to PDF and Poppler to render slide images. If they are unavailable, state the verification gap.

Review every affected slide for:

- text overflow and clipping;
- overlapping shapes, images or text;
- inconsistent margins and alignment;
- unreadable text size or low contrast;
- missing content, stale placeholders and duplicated slides.

When editing an existing deck, compare representative output slides with the source template.

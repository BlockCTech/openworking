# PPTX

Use the repository's existing presentation tooling when available. Preserve templates, speaker notes and layout conventions when editing an existing deck.

## Workflow

1. Inspect the source deck or establish the audience, message and required slide count.
2. Select an existing project library or available host tool.
3. Create or edit the deck with consistent typography, spacing and visual hierarchy.
4. Extract text to catch missing content and render slides to images for visual review.
5. Fix overflow, overlap, alignment and contrast issues before reporting completion.

Read [references/quality-checks.md](references/quality-checks.md) before final verification.
